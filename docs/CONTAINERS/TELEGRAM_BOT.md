# Telegram бот

## Описание
Telegram бот для мониторинга системы и отправки уведомлений.

## Контейнер
- **Имя**: telegram-notification-bot

## Переменные окружения
- `TELEGRAM_BOT_TOKEN` - токен бота от @BotFather
- `ADMIN_CHAT_ID` - Chat ID администратора
- `DB_HOST` - хост базы данных
- `DB_PORT` - порт базы данных
- `DB_NAME` - имя базы данных
- `DB_USER` - пользователь БД
- `DB_PASSWORD` - пароль БД
- `LOGGING_LEVEL` - уровень логирования
- `DEBUG_MODE` - режим отладки
- `HEALTH_CHECK_INTERVAL` - интервал проверки здоровья

## Volumes
- `telegram_data` - данные подписчиков
- `./config.json` - конфигурация системы
- `./database/database_manager.py` - менеджер БД

## Команды бота (доступны по клику в меню)
- `/start` - Начать работу с ботом и подписаться на уведомления
- `/status` - Получить отчет о состоянии системы
- `/myid` - Получить ваш Chat ID
- `/tank` - Состояние резервуара (уровень, объем, давление)
- `/valve` - Состояние клапана (открытие, потоки)
- `/pid` - Параметры PID контроллера (Kp, Ki, Kd, уставка)
- `/controllers` - Статус контроллеров (основной/резервный)
- `/system` - Общее состояние системы
- `/help` - Помощь по командам

## Уведомления
Все уведомления отправляются автоматически без фильтрации:
- Критические ошибки системы
- Предупреждения о проблемах
- Изменения статуса контроллеров
- Отключения контейнеров

## Особенности
- Команды доступны по клику в меню Telegram
- Автоматическая подписка при команде `/start`
- Отправка всех уведомлений (критические и предупреждения)
- Ограничение частоты: не чаще 1 сообщения в минуту для одного типа

## Health Check
```bash
python -c "import os; exit(0 if os.getenv('TELEGRAM_BOT_TOKEN') else 1)"
```

## Логи
```bash

```
