## 🔧 **Технические детали:**

### **Отслеживание состояния:**
- **Предыдущее состояние**: Сохраняется для сравнения
- **Флаг первого запуска**: `first_seen` для уведомлений о запуске
- **Счетчик перезагрузок**: Отслеживается для каждого контейнера
- **Отправленные предупреждения**: Предотвращение спама

### **Логика уведомлений:**
1. **При первом запуске**: Уведомление о запуске контейнера
2. **При отключении**: Уведомление о проблеме + попытка перезагрузки
3. **При восстановлении**: Уведомление о восстановлении + очистка предупреждений
4. **При изменении системы**: Уведомление об улучшении/ухудшении состояния

### **Предотвращение спама:**
- Используется `alerts_sent` set для отслеживания отправленных предупреждений
- Уведомления отправляются только при изменении состояния
- При восстановлении предупреждения очищаются
