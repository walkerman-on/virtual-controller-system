## 🔄 Замкнутый цикл управления

```
SP (уставка) → PID Controller → OP (клапан) → Process Model → PV (уровень) → PID Controller
```

## 📊 Параметры системы

### Модель бака
- **Высота:** 3.0 м
- **Диаметр:** 2.0 м  
- **Плотность:** 1000 кг/м³
- **Начальный уровень:** 1.5 м
- **Площадь сечения:** 3.14 м²

### Клапан
- **Kv0 (минимальный):** 0.01
- **Kvs (максимальный):** 800
- **Расчет по ГОСТ:** Q = (Kv / (3.57 × 10⁴)) × √(Δp / ρ)
- **Целевой расход при Kv100:** 160-200 м³/ч

### PID-регулятор
- **Kp:** 15.0 (оптимизирован для быстрого отклика)
- **Ki:** 0.1 (ограниченная интегральная составляющая)
- **Kd:** 1.0 (стабилизирующая производная)
- **Уставка:** 1.6 м
- **Действие:** Прямое (клапан открывается при PV > SP)

### Временные характеристики
- **Шаг симуляции:** 0.1 с
- **Частота обновления:** 10 Гц
- **Время стабилизации:** ~100-200 секунд
